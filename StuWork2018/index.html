In the Spring of 2018, we used Adobe Animate CC to create interactive animation projects. While we have done this type of interactive animation in the past, this was the first year we created them with the HTML5 Canvas and Javascript.

First we created Animated Faces, but we only published one becasue we did not start them as HTML5 Canvas projects.

Then we created Interactive Animated Cards. Each class created a Practice Card using a common storyboard, then each student had the opportunity to design their own Interactive Animated Cards.

The last interactive project we created was the Mr. Potato Head Project (MPH).
